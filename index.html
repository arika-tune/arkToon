arkToon

Hybrid NPR Rendering

単なるトゥーンシェーダーではありません。
「情報量の整理されたリッチなイラスト表現」をリアルタイム演算で実現する、
次世代のスタイライズド・ライティング・エンジン。

VCCに追加 技術的な違いを見る

なぜ「エンジン」なのか

従来のアニメ調シェーダーは「影の形を固定」していました。
arkToonは独自のスタイライズドBRDFを構築し、
光の物理挙動を「絵作り」のために再解釈して計算します。

プロシージャル・フォールオフ

巨大なランプテクスチャに依存せず、数学的な関数（Function）で影の減衰を計算します。これにより、光源が近づいた際の「白飛び」を数理的に抑制し、常に情報量の多いグラデーションを維持します。

リッチ・コンポジット

最新のRPG（原神やスターレイル等）に見られる「マルチレイヤー処理」を軽量に実装。 スペキュラ、リムライト、疑似SSSを物理ベースに近い挙動で合成し、フィギュアのような実在感を生み出します。

スマート・オプティマイズ

複雑な計算を行いつつも、テクスチャ参照数を極限まで削減。「計算はGPUにとって早いが、メモリ読み込みは遅い」という現代GPUの特性に最適化し、VRChatの大人数インスタンスでも高FPSを維持します。

Traditional Toon (lilToon / Poiyomi)

「設定した通りの色を出す」ことに特化しています。アニメの作画を完全に再現したい場合や、テクスチャで影を完全に書き込みたい場合はこちらが適しています。

arkToon (Stylized Engine)

「環境に合わせてイイ感じに見せる」ことに特化しています。ワールドの光の影響を受けつつ、リッチな質感や空気感を出したい場合、あるいはテクスチャ描画コストを下げたい場合に最強の選択肢です。

基本設定 (Base Settings)

Rendering Mode

描画モードを設定します。

  • Opaque: 不透明（標準）。最も軽量。
  • Cutout: 透過（アルファテスト）。網戸のように抜きます。
  • Transparent: 半透明。ガラスや幽霊などの表現に。

Base Map & Color

メインとなるテクスチャと乗算カラーです。アルファチャンネルはCutout/Transparentモード時の透明度として使用されます。

Normal Map

法線マップを設定します。
Is_NormalMapToBaseをONにすると、法線の凹凸がシェーディング（影の形）にも影響します。リッチな表現には必須です。

Cull Mode

ポリゴンの描画裏表設定です。マントの裏側などを描画したい場合は「OFF」にします。

影設定 (Shadow)

arkToonは「1段目の影」と「2段目の影（より暗い影）」を、ランプテクスチャではなく数値計算（Step/Feather）で生成します。

1st Shade (通常の影)

  • 1st_ShadeMap 影の範囲を指定するテクスチャ（影の焼き込み）。
  • 1st Shade Color 1段階目の影の色。
  • Step / Feather 影が入る境界線の位置と、そのぼかし具合。Featherを上げると柔らかい印象になります。

2nd Shade (濃い影)

  • 2nd Shade Color さらに光が当たらない部分の色。
  • Auto Calculation 2段目の影用テクスチャスロットはありません。1st ShadeMapが空の場合は自動計算され、設定されている場合はそのマップが流用されます。

リムライト (Rim Light)

物体の縁を光らせるリムライトを2つ搭載しています。メインのリムライトと、逆サイドの環境光的なリムを分けるなど、リッチな表現が可能です。

共通パラメーター (1st & 2nd)

Power

リムの鋭さ（範囲の狭さ）。値が大きいほど細くなります。

Feather

リムのぼかし具合。

Additive Mode

ONで「加算（発光）」、OFFで「合成（色を混ぜる）」になります。

反射と肌表現 (Spec & SSS)

Specular (天使の輪)

髪の毛のハイライトや金属光沢などに使用します。

  • Specular Map: ハイライトの強さと色を指定（RGB）。
  • TweakSpecularOnShadow: 「影の中にあるハイライト」を消す強さを調整できます。
  • Blend Mode: Replaceモードにすると、強い光が当たっても白飛びせず、指定した色でハイライトを描画します（Stylized仕様）。

Fake SSS (疑似表面下散乱)

肌の柔らかさを表現するための疑似的なSSS（Subsurface Scattering）機能です。 影の境界部分に赤み（血色）を足すことで、フィギュアっぽさを消し、生き生きとした肌を作ります。

設定方法 (重要)

1. Base OptMap を設定します。
2. OptMapの Blue (青) チャンネル を白く塗った部分にのみSSSが適用されます。
3. Skin SSS MulColor で血の色（通常は赤〜オレンジ）を設定します。

輪郭線 (Outline)

背面法線押し出し方式のアウトラインです。

基本設定

  • Width: 線の太さ。0にするとアウトラインパス自体が消え、負荷が下がります。
  • Color: 線の色。
  • Outline Map: テクスチャで線の色を指定できます。

高度な制御

  • マスク (OptMap Green)
    Base OptMapの Green (緑) チャンネル で太さを制御できます（黒=線なし、白=最大）。
  • 距離スケーリング
    カメラが近づいた時(Nearest)と離れた時(Farthest)で太さを自動調整し、遠景で線が太くなりすぎるのを防ぎます。

近接影 (Distance Overlay)

カメラがキャラクターに極端に接近した際に、視界を特定の色で塗りつぶしたり、影を落とす機能です。
VRChat等で他プレイヤーが視界に入り込んだ際の「中身が見える」現象を防いだり、ハグされた時の演出として使用されます。

表面設定 (Front)

  • Color / Intensity オーバーレイする色と、その不透明度です。通常は黒色が使われます。
  • Range (影響範囲) カメラからどのくらいの距離で効果が出始めるかを設定します。
  • Falloff Curve 減衰のカーブを「線形(Linear)」か「二次関数(Quadratic)」から選べます。

裏面・システム設定

  • Backface Darkness カメラがメッシュの内部に入り込んだ際、ポリゴンの裏面を描画して視界を遮る強さです。
  • Prevent Near Clip (ニアクリップ防止)
    頂点シェーダーでメッシュをカメラ奥（Far平面）へ押し込むことで、カメラのニアクリップ（視界がメッシュを貫通して中が見えてしまう現象）を物理的に防ぎます。
    ※VRモードでの没入感を高めるのに推奨されます。

その他 (MatCap / Emission)

MatCap (マットキャップ)

球体反射マップを使用して、金属やサテン生地のような質感を安価に追加します。
Is_LightColor_MatCapをONにすると、ワールドのライトカラーと影の影響を受け、暗闇でMatCapだけが浮くのを防げます。

Emission (発光)

暗闇で光る部分を設定します。HDRカラー対応。
Emission Mapを設定することで、特定の部分だけを光らせることができます。
