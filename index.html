Core Logic

「ちょうどいい」を科学する、
3つのレンダリング哲学。

01

Procedural Falloff

汚くない影。
単純な閾値だけでなく、光の減衰をプロシージャルに計算。ローポリゴン特有の影のガタつきを滑らかに補間し、イラストの平面性と3Dの立体感を高次元で両立させます。

02

Rich Composite

コンポジット済みの質感。
ダブルリムライト、スペキュラ、疑似SSS。撮影処理（After Effects等）で行うようなレイヤー合成をシェーダー内で完結。VRChat上で「撮影済みの画」をリアルタイムに出力します。

03

Smart Optimize

賢い軽量化。
「見た目に寄与する計算」だけを厳選し、不要なレンダリングパスを徹底排除。リッチな質感を実現しながら、イベント会場でも耐えうる驚異的な軽量動作を維持しています。

まずはここから

モデルの基本的な描画方法を決定します。「プリセット」機能を使えば、面倒な数値調整を一瞬で終わらせることができます。

Rendering Mode
  • Opaque (不透明): 肌や服など、透けないパーツ用。最速。
  • Cutout (切り抜き): メッシュ形状が複雑な衣装のレースなど。
  • Transparent (半透明): 透けるヴェールや頬染めチークなど。
便利機能: Presets

GUI上部のボタンで一発設定できます。

  • Figure スッキリとした現代風3DGC調
  • Deep Shadow 重厚感のある劇画調
Pro Tip: 白飛び対策

arkToonには「Hi-Cut Filter」が搭載されています。 ワールドのライトが眩しすぎる（HDR強度が1.0を超える）場合でも、自動的に色味を維持したまま輝度を抑え、顔が真っ白になるのを防ぎます。

※設定不要で自動適用されます。

アニメ塗りの命「2段階の影」

arkToonでは「薄い影（1st）」と「濃い影（2nd）」を個別に設定できます。 これにより、単調になりがちな3Dモデルの影に「イラストのような情報量」を与えます。

1st
1st Shade (基本の影)

全体の立体感を担当します。Feather(ぼかし)を少し広め（0.3〜0.5）に取ると、ローポリゴンのカクつきを目立たなくさせる効果があります。

2nd
2nd Shade (最も暗い影)

服の重なりや、光が全く届かない場所を担当します。範囲を狭く、境界をくっきりさせると画面が引き締まります。

色選びのコツ

影の色に「完全な黒」や「グレー」を使うと、モデルがくすんで見えてしまいます。

・肌の影 → 薄いピンクやオレンジ
・白服の影 → 薄い青紫

このように「彩度のある色」を設定すると、リッチな仕上がりになります。

背景からキャラを際立たせる

逆光表現（リムライト）です。arkToonは2つのリムライトを持てるため、 「上からの強い光」と「下からの環境光の照り返し」を同時に表現できます。

Addtive Mode (加算) の使い分け
ON (発光)

金属、濡れた質感、強い日差し。色が重なり合って白く輝きます。

OFF (合成)

布、肌、柔らかい照り返し。白飛びせず、指定した色がそのまま乗ります。

Recommended Setup

1st Rim: Additive ON

色: 白〜薄い黄色 / Power: 強め

2nd Rim: Additive OFF

色: 空の色や環境色 / Power: 弱め

この組み合わせで「空気感」のあるリッチな画になります。

質感を「嘘」で演出する

Specular (ハイライト)

光の反射を描画します。v0.1.3で追加されたBlend Modeが強力です。

  • Replace Mode (新規)

    ハイライトの色で下地を「置き換え」ます。加算合成しないため白飛びせず、髪の天使の輪や、アニメ調のくっきりしたハイライトに最適です。

  • Additive Mode (従来)

    光を加算します。ラテックス、金属、宝石など、ギラギラ輝かせたいパーツに向いています。

Fake SSS (疑似表面下散乱)

光と影の境界線（明暗の境目）にだけ、指定した色（赤みなど）を乗せる機能です。 「光が皮膚を透過して赤く見える現象」を擬似的に再現し、肌の柔らかさを表現します。

重要

全身にかかると服まで赤く光ってしまいます。
必ずOptMapのBlue(青)チャンネルで、肌の部分だけが白くなるようなマスクを指定してください。

崩れない輪郭線

背面法線方式のアウトラインです。arkToonのアウトラインは、カメラとの距離に応じて太さを自動調整する機能を持っています。

Distance Scaling (距離補正)

遠くに離れた時、輪郭線が太いままだとモデルが黒く潰れてしまいます。
Farthest Distance（最長距離）を設定することで、遠景では自動的に線を細くし、美しい見た目を維持します。

Masking (マスク制御)

口の中、瞳、髪の毛先など、線を出したくない場所は OptMapのGreen(緑)チャンネルを黒く塗ることで、部分的に線を消すことができます。

フィギュア風設定

最近のトレンドとして、あえて輪郭線を「0」にする、または極細にして色を肌馴染みの良い色（茶色など）にすると、フィギュアのような繊細な質感が出せます。

VR接写の「お作法」

VRChatなどのVR空間では、他プレイヤーが極端に近づく（撫でる、ハグする）ことがあります。 その際、カメラがモデルを突き抜けて中身が見えてしまうのを防いだり、接近した際の違和感を防ぐのが「Distance Shadow（近接影）」です。

  • Prevent Near Clip (ニアクリップ防止)

    カメラが顔に埋まった瞬間、ポリゴンが消失してしまうのを防ぎます。

  • Backface Opacity (裏面の塗り)

    カメラが突き抜けてしまった場合でも、裏面を「メインテクスチャの平均色」で塗りつぶして表示します。 中身が透明な空洞に見えるホラー現象を回避できます。
